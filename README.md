# Webshop project

1. Clone Repo
----------------------- CD INTO CLONED REPO
2. Run 'composer install'
3. Run 'npm install'
4. Create a DB and set your own .env (you can use the example file to see what is needed)
5. Run 'php artisan migrate --seed'
6. Enjoy the website!
